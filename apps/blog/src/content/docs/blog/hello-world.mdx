This is the first post on the new Lightforge blog. The platform shipped
this week — Astro Starlight, edge-routed at `lightforge.dev/blog`,
auto-deployed via Cloudflare Pages, with `llms.txt` and per-page Markdown
view for agent consumers.

More on the architecture, the editorial gates, and the publish-blog
agent tool — designed against [ADR-086 (Irreversibility Contract)](https://docs.lightforge.dev/governance/adr-086-irreversibility-contract/) — in upcoming posts.
